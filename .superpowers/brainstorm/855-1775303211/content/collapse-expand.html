▶ 3.1 What is a Process?

A process is an instance of a program in execution. It includes the program code, current activity (program counter), stack, data section, and heap.

Definition: A process is the unit of work in most systems.
▶ 3.2 Process States

Processes transition between states: New → Ready → Running → Waiting → Terminated. The OS scheduler manages these transitions.
