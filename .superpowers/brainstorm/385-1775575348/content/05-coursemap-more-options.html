Pros

  • Visually striking — feels like a game level map
  • Very compact vertically (4 per row)
  • Green "filling" path gives strong sense of progress
  • Mobile: naturally wraps to fewer per row

Cons

  • Zigzag reading order is less intuitive than left-to-right
  • No titles visible without hover
  • Implementation is more complex (S-curve layout)
