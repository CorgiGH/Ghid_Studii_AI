Binary Search

Key decisions:
• Labels always below — consistent scanning, no collision with connectors above
• Connectors route along the outer edge with a rounded corner — clears the label zone
• Short labels (1-2 words, truncated) — full title shown in detail card on hover/tap
• Active node label uses accent color + bold to stand out

Option D2: Alternating Above/Below

Gemini's suggestion: odd rows have labels above, even rows below
