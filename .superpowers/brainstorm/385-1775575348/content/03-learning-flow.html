Quiz Feedback: Targeted Explanation (No Retry)

Wrong answer → instant visual + per-distractor explanation + optional deep dive

Before answering
What is the time complexity of binary search?
A) O(n)
B) O(log n)
C) O(n log n)
D) O(1)
After wrong answer (chose A)
What is the time complexity of binary search?
A) O(n) ✗
B) O(log n) ✓
C) O(n log n)
D) O(1)
Why A is wrong
O(n) is the complexity of linear search, which checks every element. Binary search halves the search space at each step, giving O(log n).
Why B is correct
Each comparison eliminates half of the remaining elements. For n elements, you need at most log2(n) comparisons.
Review this in Step 4: Binary Search →
UX flow: 1. Student selects wrong answer → instant red/green highlights (no delay)
2. Per-distractor explanation appears: "Why A is wrong" + "Why B is correct" (2-3 sentences each)
3. Optional "Review this in Step N" link for deeper context
4. No retry — move forward. Wrong questions re-appear in end-of-course review quiz.

JSON schema: Each quiz option gets an optional explanation field (bilingual). The pipeline generates per-distractor explanations + a reviewStep reference.
